CSS简介

CSS （Cascading Style Sheets）是层叠样式表用来定义网页的显示效果。可以解决html代码对样式定义的重复，提高了后期样式代码的可维护性，并增强了网页的显示效果功能。简单一句话：CSS将网页内容和显示样式进行分离，提高了显示功能。
层叠：是指类似于刷油漆，一层一层嵌套
样式：指有很多属性值
好处：实现网页内容和显示样式分离，便于维护

使用@import引入外部CSS文件

我们可以调用外部写好的CSS文件，而不用我们在HTML文件中自己添加相关样式
优点：便于维护，可以导入别人的CSS文件
缺点：有些浏览器不支持

CSS的盒子模型

盒子的边框的各种配置，比如颜色，粗细，虚实
