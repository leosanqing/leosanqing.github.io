js的String对象

js的String 主要分为两种，一种是可以操作html的方法，一种是和java相似的方法，下面我从这两个大的方向解释js的String

与html相关的方法

这个主要是修改网页上文字的显示效果，例如bold()将文字加粗，fontcolor()修改字体的颜色；fontsize()字体大小；link（）将其设置为超链接演示代码：

与Java相似的方法

charAt() 返回相应位置的字符（从0开始）
indexOf()返回相应字符的索引（从0开始，且是第一个）
replace()替换相应位置的字符,并返回替换后的字符串
split()按照相应的规则拆分字符串

JS-array

array的创建

JS里面的array和Java里面的array非常相似，只不过在我们创建数组的时候并没有java那么严格设定类型，比如创建Java数组时 int arr[]=new int [10];,之前说过 JS中并没有像Java中的那样，他是弱类型语言，并没有那么严格的设定，因此，他的数组创建为var arr[]=new Array(8);，里面可以放任何类型的元素，这点和java不一样，但是java也可以实现相应的方法，比如我们创建数组时可以这样创建Object arr[]=new Object[10];,因为我们知道Java中Object是所有类父类，因此这里面也可以放任意类型的元素；

与java一样，JS创建数组时也有三种方式，

array的方法

push()

在数组末尾添加新元素，并返回数组长度

pop()

删除数组末尾的元素，并返回删除的元素

reverse()

将数组里面的元素从后到前输出,但这个不是java里面的回文，他只是将数组中的元素，从后往前输出，每个元素里面的顺序并不不会倒着输出；

JS-Date

JS操作时间的方法

toLocaleString

将时间格式化为本地时间格式，请注意这里是toLocaleString，里面还有一个e

getFullYear()

获取完整年份信息

getMonth()

获取月份，但是是从 0开始算

getDay()

获取一周的星期几

getDate()

获取一个月的第几天，注意与 getDay()方法分开

获取小时、分、秒

getHours()
getMinutes()
getSeconds()
请注意，每个后面都有s

getTime()

获取从1970-1-1 0:0:0至今的毫秒数

JS-Math

这些是JS中与数学有关的方法

ceil()

向上舍入，比如现在有一个小数，1.12；用这个方法得到的结果就是2

floor()

向下舍入，比如现在有一个小数，1.8，不管小数位后面多大，都舍去，结果为1

round()

四舍五入
比1.5大的向上取，比1.5小的小数舍去

random()

随机数,[0,1)之间的任意小数

JS-全局函数

eval()

执行代码，如果你的字符串是合理的且符合规范的代码，那么将执行这行代码

encodeURI()

对其进行编码，因为有的并不支持比如中文之类的文字，会把其进行编码以后上传

不进行编码的文字

decodeURI()

对其进行解码

JS-BOM

JS的BOM对象是(Borswer Object Model)浏览器对象模型，主要是操作浏览器对象的相关操作

Navigator

浏览器本地信息，比如想知道使用的浏览器名字，则可以使用appName()方法

screen

获取屏幕的信息，比如宽、高、分辨率

location

获取当前页面的地址，也可以设置地址

history

实现比如返回上一个页面，下一个页面

window

窗口对象，顶层对象，是之前所有对象的父对象，所有的BOM对象都可以从window对象中找到

window方法

alert()弹出提示框，之前应该看到过
confirm()弹出确认窗口，类似我们删除文件时的对话框
prompt()弹出输入框，但是基本废弃，因为窗口太丑
open()打开一个窗口，三个参数，现在只需要知道两个参数，第一个是新开地址的URL，第三个是新开窗口的长宽高"width=15,height=123"

setInterval()每隔一段相应的，执行一次，有两个参数，第一个是执行的代码，第二个是毫秒数
比如我们要每隔3秒系统提示一次已经过了三秒

这样每隔三秒就会提示已经过了三秒

setTimeout()和上一个类似，只是只执行一次

clearInterval()可以清除之前设置的 Interval代码
